Сервер будет использоваться для хранения файлов сотрудников и работы с общими папками по протоколу SMB. Требуется организовать работу с файловой системой NFTS, являющейся разработкой Microsoft. Эта файловая система используется семейством ОС Windows и Windows 10 в частности, и сервер с ОС Windows Server 2022 обеспечит стабильную работу с компьютерами пользователей благодаря совместимости систем.
